# MiniTag 迷你版标签

## 何时使用
- 需要缩小版的标签时

## 代码演示

```js
import MiniTag from "@/components/MiniTag";

const App: React.FC = () => {
  return (
    <MiniTag>迷你版标签</MiniTag>
  );
}

```

## API

| 参数 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| scaleValue | 缩小值 | number | 0.85 |
[其他参数见](https://ant.design/components/tag-cn/#API)
