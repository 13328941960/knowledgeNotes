## API

| 参数 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| text | 提交按钮文案 | string \| number  | - |
| isToFixed2 | 是否保留两位小数 | boolean | - |
| isBold | 是否加粗 | boolean | - |
